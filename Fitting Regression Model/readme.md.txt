In this project, I tried to fit a regression model to predict the miles per gallon of different car. For this project, I have chosen the from Kaggle website, I did some analysis using scatter plots, then fit Linear Regression, Lasso Regression and Ridge Regression and compared the accuracy of them. 

Software Used : Python (Libraries: Pandas, Numpy, Matplotlib, sklearn), Jupyter Notebook
